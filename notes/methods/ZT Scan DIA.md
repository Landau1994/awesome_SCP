# ZT Scan DIA

**A scanning quadrupole DIA approach designed for improved quantification performance.**

## Overview

ZT Scan DIA is a modern DIA acquisition mode referenced in benchmarking studies for short-gradient and low-input workflows.

## Key Features

- **Scanning Acquisition**: Uses dynamic quadrupole scanning behavior.
- **Quantification Benefit**: Reported improvements in accuracy under challenging conditions.
- **Benchmark Relevance**: Included in multi-platform performance evaluations.

## Workflow

```text
Scanning quadrupole DIA acquisition → DIA feature extraction → Quantification and benchmarking
```

## Performance

- **Proteins per cell**: Competitive for low-input benchmarks
- **Throughput**: High with short gradients
- **Data completeness**: Good in optimized platform settings

## Advantages

- Suitable for rapid, large-cohort acquisitions
- Improved quantification in some low-input contexts
- Pairs well with modern DIA software

## Limitations

- Platform availability constraints
- Requires instrument-specific method tuning
- Comparative performance can be dataset dependent
