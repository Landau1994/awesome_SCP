# nanoLC-MS

**Nano-flow liquid chromatography coupled with mass spectrometry for high-sensitivity proteomics.**

## Overview

nanoLC-MS is a standard high-sensitivity platform in proteomics, widely used for low-input and single-cell-adjacent sample analysis.

## Key Features

- **High Sensitivity**: Nano-flow improves ionization efficiency.
- **Low Sample Requirement**: Suitable for limited material.
- **Broad Compatibility**: Works with DDA/DIA and multiple instruments.

## Workflow

```
Sample prep → nanoLC separation → Electrospray ionization → MS/MS acquisition
```

## Performance

- **Proteins per cell**: Workflow- and instrument-dependent
- **Throughput**: Moderate to high (short-gradient setups)
- **Data completeness**: High with optimized gradients/software

## Advantages

- Strong sensitivity for low-abundance peptides
- Established ecosystem of methods and tools
- Flexible for discovery and targeted workflows

## Limitations

- Susceptible to clogging/carryover if not maintained
- Method optimization can be instrument-specific
- Throughput trade-off versus deeper gradients
