# Activity-based protein profiling

## Overview

ABPP uses activity-based probes to profile functional enzyme states rather than total abundance.

## Key Features

- Captures active protein fractions
- Supports target engagement studies
- Useful in drug discovery and chemoproteomics

## Workflow

```text
Probe labeling → Enrichment/detection → LC-MS analysis → Activity-centric interpretation
```
