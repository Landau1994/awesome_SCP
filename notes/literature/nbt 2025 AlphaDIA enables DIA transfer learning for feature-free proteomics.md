# AlphaDIA enables DIA transfer learning for feature-free proteomics

## Abstract

The scale of data generated for mass-spectrometry-based proteomics and modern acquisition strategies poses a challenge to bioinformatic analysis. Search engines need to make optimal use of the data for biological discoveries while remaining statistically rigorous, transparent and performant. Here we present alphaDIA, a modular open-source search framework for data-independent acquisition (DIA) proteomics. We developed a feature-free identification algorithm that performs machine learning directly on the raw signal and is particularly suited for detecting patterns in data produced by time-of-flight instruments. Benchmarking demonstrates competitive identification and quantification performance. While the method supports empirical spectral libraries, we propose a search strategy named DIA transfer learning that uses fully predicted libraries. This entails continuously optimizing a deep neural network for predicting machine-specific and experiment-specific properties, enabling the generic DIA analysis of any post-translational modification. AlphaDIA provides a high performance and accessible framework running locally or in the cloud, opening DIA analysis to the community.

## Key Points

- Introduces [[alphaDIA]], an open-source, modular, and feature-free search framework for Data-Independent Acquisition (DIA) proteomics.
- Employs a machine learning approach that operates directly on raw signals using convolution kernels, avoiding the information loss typically associated with centroiding and feature boundaries in time-of-flight (TOF) data.
- Pioneers "DIA transfer learning," a method that fine-tunes generic deep learning models on experiment-specific data to improve the prediction of retention times, MS2 spectra, and charge states, particularly effective for identifying unseen post-translational modifications (PTMs).

## Results

### Main Findings
1. **Feature-Free Processing Adaptability**: AlphaDIA's feature-free processing successfully processes data across various DIA modes (e.g., dia-PASEF, synchro-PASEF, SWATH, variable window DIA) and instrument platforms (Bruker, Sciex, Thermo Orbitrap/Astral), matching or exceeding the identification depth of established commercial and open-source tools.
2. **High-Performance Predicted Library Search**: Using fully predicted spectral libraries generated by [[alphaPeptDeep]], alphaDIA identified over 120,000 precursors and 9,800 protein groups in a 60-SPD method on the Orbitrap Astral, with rigorous false discovery rate (FDR) control validated via Arabidopsis entrapment.
3. **Efficacy of DIA Transfer Learning**: Transfer learning enabled the framework to adapt on-the-fly to specific PTMs (e.g., dimethylation), drastically reducing median retention time prediction error from 317s to 11s and increasing unique precursor identifications by 48% and protein groups by 25%.

## Discussion

### Strengths
- Avoids signal loss and boundary errors in stochastic TOF data through its continuous, feature-free scoring strategy.
- Open-source, Python-based modular architecture allows transparent development and easy integration of new algorithms.
- DIA transfer learning eliminates the need for generating expensive empirical libraries or specifically pre-trained models for novel and complex PTMs.

### Limitations
- Explicitly modeling the ion mobility dimension for massive fully predicted libraries leads to increased processing times (e.g., >1 h per file), requiring future optimizations.

### Future Work
- Improving processing speed and computational efficiency for large-scale predicted libraries with ion mobility data.
- Establishing alphaDIA as a foundational, community-driven framework tightly coupled to ongoing advancements in artificial intelligence for proteomics.
