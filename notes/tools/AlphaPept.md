# AlphaPept

**An open-source proteomics framework for MS data processing and analysis.**

## Overview

AlphaPept is an analysis environment for peptide/protein identification and quantification workflows, with an emphasis on accessibility and extensibility.

## Key Features

- **Open-Source Workflow**: Transparent and extensible analysis steps.
- **Proteomics Pipeline Support**: Handles core processing tasks.
- **Community-Oriented**: Integrates with modern ML-enabled ecosystem.
- **Cross-Platform**: Windows, Linux, macOS
