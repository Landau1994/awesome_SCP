# Proteome Discoverer

**A Thermo Scientific software suite for proteomics data identification and quantification.**

## Overview

Proteome Discoverer provides an integrated environment for processing DDA/DIA proteomics datasets, including search workflows and result reporting.

## Key Features

- **Integrated Workflow UI**: Modular processing and reporting.
- **Vendor Ecosystem Support**: Tight integration with Thermo instruments.
- **Extensible Nodes**: Supports multiple search/validation components.
- **Cross-Platform**: Windows, Linux, macOS
