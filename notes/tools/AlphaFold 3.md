# AlphaFold 3

**A structure prediction system for modeling biomolecular complexes.**

## Overview

AlphaFold 3 is used for structural hypothesis generation and can support interpretation in proteoform and interaction-focused studies.

## Key Features

- **Complex-Level Prediction**: Models interactions beyond single proteins.
- **Structure-Guided Insight**: Supports mechanistic interpretation.
- **Broad Biomolecular Scope**: Useful in multi-component systems.
- **Cross-Platform**: Windows, Linux, macOS
