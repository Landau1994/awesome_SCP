# FragPipe

**Integrated proteomics analysis platform for identification and quantification workflows.**

## Overview

FragPipe provides a unified interface and workflow ecosystem for MS-based proteomics, often used for peptide/protein identification and library generation.

## Key Features

- **Integrated Pipeline**: Brings multiple proteomics engines into one workflow.
- **Library/ID Support**: Useful for both discovery and targeted analysis.
- **User-Friendly Operation**: Designed for practical analysis at scale.
- **Cross-Platform**: Windows, Linux, macOS
