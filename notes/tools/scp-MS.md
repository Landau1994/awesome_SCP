# scp-MS

**A shorthand term for single-cell proteomics mass spectrometry workflows.**

## Overview

scp-MS generally refers to MS-based single-cell proteomics acquisition and analysis pipelines rather than a single software package.

## Key Features

- **Single-Cell Focus**: Designed for low-input cell-level proteome readout.
- **Workflow Umbrella Term**: Covers method + instrumentation + analysis stack.
- **Multi-Omics Compatible**: Often integrated with transcriptomics.
- **Cross-Platform**: Windows, Linux, macOS
