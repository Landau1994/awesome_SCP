# ImageJ

**A foundational open-source image processing platform for scientific data.**

## Overview

ImageJ is widely used for microscopy image visualization, quantification, and basic/advanced processing in bioimaging pipelines.

## Key Features

- **General Image Processing**: Core filtering, measurement, and ROI tools.
- **Extensibility**: Macro and plugin support.
- **Scientific Adoption**: Longstanding use in biology and pathology.
- **Cross-Platform**: Windows, Linux, macOS
