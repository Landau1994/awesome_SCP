# Fiji

**An ImageJ distribution with plugins for biological image analysis.**

## Overview

Fiji ("Fiji Is Just ImageJ") is a common environment for microscopy preprocessing, segmentation support, and quantitative image measurements.

## Key Features

- **Plugin-Rich Ecosystem**: Extensive image analysis extensions.
- **Microscopy-Oriented**: Strong support for biological imaging workflows.
- **Scripting Support**: Automates repetitive analysis steps.
- **Cross-Platform**: Windows, Linux, macOS
