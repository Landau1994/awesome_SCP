# Tecan UNO

**Automation platform used for sample preparation in proteomics workflows.**

## Overview

Tecan UNO is referenced as an automated liquid-handling setup for reproducible preparation of low-input proteomics samples.

## Key Features

- **Automation**: Reduces manual variability in sample prep.
- **Reproducibility**: Supports standardized protocol execution.
- **Low-Input Friendly**: Useful in limited-sample/single-cell contexts.
- **Cross-Platform**: Windows, Linux, macOS
