# Scanpy

**A Python toolkit for scalable single-cell omics analysis and visualization.**

## Overview

Scanpy is widely used for preprocessing, clustering, embedding, and trajectory analysis in single-cell studies, including integrated SCP analyses.

## Key Features

- **Single-Cell Workflow**: End-to-end analysis utilities.
- **Scalable Processing**: Supports large datasets through sparse structures.
- **Visualization Tools**: UMAP/t-SNE and marker exploration.
- **Cross-Platform**: Windows, Linux, macOS
