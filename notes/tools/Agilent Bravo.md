# Agilent Bravo

**An automated liquid-handling platform for reproducible sample preparation workflows.**

## Overview

Agilent Bravo is used for high-throughput and standardized proteomics sample processing in large-cohort studies.

## Key Features

- **Automation**: Reduces manual pipetting variability.
- **Scalable Throughput**: Supports larger sample batches.
- **Protocol Reproducibility**: Improves inter-run consistency.
- **Cross-Platform**: Windows, Linux, macOS
