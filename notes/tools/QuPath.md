# QuPath

**An open-source platform for digital pathology image analysis.**

## Overview

QuPath is used for tissue image processing, segmentation, and annotation, often integrated with spatial proteomics workflows.

## Key Features

- **Whole-Slide Analysis**: Handles large pathology images.
- **Segmentation & Annotation**: Cell/object detection and labeling.
- **Interoperability**: Supports export for downstream computational analysis.
- **Cross-Platform**: Windows, Linux, macOS
