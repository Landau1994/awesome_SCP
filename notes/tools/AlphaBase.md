# AlphaBase

**A foundational data/model utility layer used in AlphaPept ecosystem workflows.**

## Overview

AlphaBase supports peptide/proteomics data representations and shared utilities for deep-learning-enabled proteomics tools.

## Key Features

- Shared infrastructure for model pipelines
- Peptide-centric data handling support
- Integration with AlphaPeptDeep workflows
