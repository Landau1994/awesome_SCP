# PEAKS

**A proteomics software platform for peptide identification, de novo sequencing, and quantification.**

## Overview

PEAKS is a commercial proteomics package widely used for peptide identification and advanced search tasks, including de novo-assisted workflows.

## Key Features

- **De Novo Support**: Facilitates sequence inference without full database reliance.
- **Integrated Search**: Covers identification and quantification in one environment.
- **Workflow Versatility**: Used across DDA/DIA scenarios.
- **Cross-Platform**: Windows, Linux, macOS
