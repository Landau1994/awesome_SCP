# Skyline

**A widely used platform for targeted proteomics method development and quantitative analysis.**

## Overview

Skyline is commonly used for PRM/SRM method setup, chromatogram inspection, and targeted peptide quantification in translational proteomics workflows.

## Key Features

- **Targeted Quantification**: Supports SRM/PRM style workflows.
- **Method Building**: Facilitates transition list and assay development.
- **QC-Friendly**: Strong visualization and validation utilities.
- **Cross-Platform**: Windows, Linux, macOS
