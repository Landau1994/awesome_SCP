# DirectLFQ

**A label-free quantification strategy/tool for robust protein-level intensity summarization.**

## Overview

DirectLFQ provides normalization and protein quantification from peptide-level evidence in label-free proteomics workflows.

## Key Features

- **Label-Free Quantification**: Supports LFQ protein abundance estimation.
- **Normalization Support**: Improves cross-sample comparability.
- **Workflow Integration**: Used in modern DIA-oriented analyses.
- **Cross-Platform**: Windows, Linux, macOS
