# Perseus

**A proteomics data analysis platform for statistics, annotation, and interpretation.**

## Overview

Perseus is used for downstream proteomics matrix analysis including filtering, normalization, statistical testing, and functional interpretation.

## Key Features

- **Matrix-Centric Analysis**: Designed for quantitative proteomics tables.
- **Built-in Statistics**: Common tests and visual exploration.
- **Biological Interpretation**: Annotation and enrichment utilities.
- **Cross-Platform**: Windows, Linux, macOS
