# Prosit

**A deep-learning system for predicting peptide tandem mass spectra and retention properties.**

## Overview

Prosit is a widely used prediction tool for spectral library generation and search enhancement in proteomics analysis.

## Key Features

- **Spectrum Prediction**: High-quality in silico fragment intensity prediction.
- **Library Augmentation**: Supports DIA/DDA identification workflows.
- **ML-Driven Performance**: Improves coverage in challenging datasets.
- **Cross-Platform**: Windows, Linux, macOS
