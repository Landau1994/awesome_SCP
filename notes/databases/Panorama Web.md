# Panorama Web

## Overview

Panorama Web is a repository and sharing platform for targeted proteomics assays and Skyline projects.

## Data Type

- Targeted proteomics method files
- Quantitative assay outputs

## Access

- Web interface for project browsing and download

## Use Cases

- Sharing and reusing targeted assay setups
- Benchmark reproducibility and method transfer
- Collaborative curation of targeted analyses
