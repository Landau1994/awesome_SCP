# SingPro

**Single-cell Proteomics Knowledge Base**

## Overview

SingPro is a specialized database for single-cell proteomics that uniquely provides raw data access and detailed experimental metadata for both MS-based and flow cytometry-based studies.

## Key Features

- **Raw Data Access**: Original data files available
- **Experimental Details**: Comprehensive protocol information
- **Expression Profiles**: Processed protein quantification
- **Multi-Technology**: MS and flow cytometry coverage

## Data Content

### Metadata Fields
- Sample type and organism
- Cell isolation method
- MS instrument parameters
- Data processing pipeline
- Quality metrics

### Expression Data
- Protein-level quantification
- Cell-level annotations
- Batch information
