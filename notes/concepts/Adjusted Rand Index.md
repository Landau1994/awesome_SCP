# Adjusted Rand Index

## Summary

Adjusted Rand Index (ARI) quantifies clustering agreement while correcting for random chance.

## Notes

- Commonly used to compare predicted clusters against reference labels.
- Values near 1 indicate strong agreement.

## References

- Standard clustering evaluation metric.
